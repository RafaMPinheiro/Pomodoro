# Welcome to Pomodoro 👋
![Version](https://img.shields.io/badge/version-0.1.0-blue.svg?cacheSeconds=2592000)
[![Documentation](https://img.shields.io/badge/documentation-yes-brightgreen.svg)](https://github.com/RafaMPinheiro/Pomodoro.git)

O Pomodoro Timer é uma técnica de gerenciamento de tempo que foi desenvolvida por Francesco Cirillo no final da década de 1980. O nome &#34;Pomodoro&#34; vem do fato de Cirillo ter usado um cronômetro de cozinha em forma de tomate para medir os períodos de trabalho.

## Install

```sh

```

## Usage

```sh

```
